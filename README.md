# Cast & Response

A podcast player that lets you react to episodes with timestamped emoji reactions and comments.

## Features

- Load any podcast via RSS feed URL
- Save your favorite podcasts for quick access
- Add emoji reactions at specific timestamps while listening
- Add optional comments to your reactions
- Click reactions to jump to that moment in the episode
- Visual markers on the progress bar show where you've reacted
- All reactions persist locally in your browser

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
